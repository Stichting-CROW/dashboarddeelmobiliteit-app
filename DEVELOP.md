# Developing the Dashboard Deelmobiliteit app

Interested in developing this app further? We love that!

- Get to know the [software architecture](https://docs.dashboarddeelmobiliteit.nl/start/architecture/)
- Read the docs on [how to install](https://docs.dashboarddeelmobiliteit.nl/start/how_to_install/) the full stack on your own server.
- Suggest features and report bugs using the [Issues](https://github.com/Stichting-CROW/dashboarddeelmobiliteit-app/issues) page

Want to get started quickly? Send an email to [info@dashboarddeelmobiliteit.nl](mailto:info@dashboarddeelmobiliteit.nl) and give you a head start in an online meeting.
